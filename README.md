FPGA Smart Surveillance System

Overview

This project is a real-time smart surveillance system developed using the PYNQ-Z2 FPGA board, Python, and OpenCV. The system captures live video through a webcam and performs surveillance-based monitoring and processing.

Features

- Real-time video surveillance
- Webcam integration
- FPGA-based processing
- Python and OpenCV implementation
- Live monitoring system

Technologies Used

- PYNQ-Z2 FPGA Board
- Python
- OpenCV
- Jupyter Notebook

Files Included

- "PYNQ_Z2_Smart_Surveillance.ipynb" – Main project notebook
- "report_fpga.pdf" – Project report
- "FPGA1 finalreport.pptx" – Project presentation

Hardware Requirements

- PYNQ-Z2 FPGA Board
- USB Webcam
- Ethernet Cable
- SD Card with PYNQ Image
- Power Adapter

Software Requirements

- PYNQ Framework
- Python Libraries
- OpenCV
- Jupyter Notebook

Project Workflow

1. Connect webcam to FPGA board
2. Connect FPGA board to laptop using Ethernet
3. Power ON the FPGA board
4. Access Jupyter Notebook
5. Run surveillance notebook
6. Monitor live video output
7. detects and Sends the notification to the Telegram 

Applications

- Smart Surveillance
- Security Monitoring
- Real-time Video Processing
- Embedded Vision Systems
